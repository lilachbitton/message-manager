# 📨 מנהל מסרים שיווקיים

אפליקציה לניהול ומעקב אחר מסרים שיווקיים לאתגרים חודשיים.

## תכונות

- 📊 **לוח מסרים** - תצוגת ציר זמן של האתגר הפעיל
- 📋 **טמפלייטים** - מאגר טמפלייטים מאורגן לפי "ימים יחסיים לאתגר"
- 🔗 **לינקים מקוצרים** - מאגר לינקים עם הוספה מהירה למסר
- ⚙️ **הגדרות אתגר** - בחר תאריך וכל המסרים ישובצו אוטומטית

## הרצה מקומית

```bash
npm install
npm run dev
```

## בנייה לפרודקשן

```bash
npm run build
```
